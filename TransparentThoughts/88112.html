浅尝Groovy：GroovyMarkup- -

                                      

GroovyMarkup可以很方便地生成XML标记串，我觉得是最方便的一种方式了，比模板填充、Castor映射等方式都要直观易懂。譬如一个Person类：
 
class Person {
 _name;
 _age;
 _sex;
...
}
 
下列方法可以生成XML标记串：
 
import groovy.xml.MarkupBuilder
class Person {
...
 toXml() {
  xml = new MarkupBuilder();
  xml.person(name:_name){
   age(_age)
   sex(_sex)
  }
 }
}
 
调用toXml()方法，得到下列字符串：
 
<person name='Gigix'>
  <age>24</age>
  <sex>male</sex>
</person>
