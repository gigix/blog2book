CCMon 0.0.1- -

                                      

http://code.google.com/p/ccmon

It's -- yet another -- CruiseControl monitor, working as a Google Desktop plugin (a.k.a. "gadget"). You can try it now:

1. check source code out via svn. (see http://code.google.com/p/ccmon/source)
2. run rake build.
3. install build/dist/ccmon.gg
4. config it in "Options..."

Enjoy! (And raise hands to push it forward.)
