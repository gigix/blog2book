伟大的Hello World- -

                                      

Hello World绝对是世界上最伟大的程序。
 
实现一个Hello World其实意味着很多。首先，这意味着你知道怎么把电脑开起来，并且知道程序是一个可执行的指令序列，对于确定的输入可以给出确定的输出。然后，实现Hello World意味着你知道如何配置一个开发环境和运行环境，意味着你知道如何处理输入和输出，意味着你知道如何用一种语言在输入/输出之间做（哪怕是最简单的）转换。剩下的就是设计模式和业务的问题了，语言本身的障碍不复存在。
 
譬如这个程序：
public String sayHello(String name) {
  return "Hello, " + name;
}
 
我们的任何一个业务方法难道不都跟它一样吗？只不过是转换的规则不同而已。
 
（亲手写出有生以来第一个EJB有感）
